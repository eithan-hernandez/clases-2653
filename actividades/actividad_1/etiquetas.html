Etiquetas de texto

Las etiquetas de texto nos ayudan a poder comunicar ideas y poder maquetar correctamente todo lo que queremos mostrar en la pagina web

  • Etiqueta HTML < b >: Para texto en negrita.
  • Etiqueta HTML < em >: Para enfatizar texto.
  • Etiqueta HTML < i >: Para texto en cursiva.
  • Etiqueta HTML < u >: Para texto subrayado.
  • Etiqueta HTML < strong >: Para textos importantes.
  • Etiqueta HTML < big >: Texto grande.
  • Etiqueta HTML < small >: Para texto pequeño.
  • Etiqueta HTML < sub >: Para texto subíndice.
  • Etiqueta HTML < sup >: Para texto superíndice.
  • Etiqueta HTML < ins >: Para texto insertado.
  • Etiqueta HTML < del >: Para texto borrado.
  • Etiqueta HTML < mark >: Para texto marcado.

Por otro lado cabe mencionar que existe una diferencia entre las etiquetas strong y bold que tenemos que tener presente

  1. < b > define texto en negrita, sin ninguna importancia adicional.
  2. < strong > define texto fuerte, con importancia semántica aumentada.

De igual forma podermos ver que ocurre un caso similar con las etiquetas i y em

  1. < i > define el texto en cursiva, sin ninguna importancia adicional.
  2. < em > define el texto enfatizado, con una importancia semántica aumentada.
